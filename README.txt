CRUD Implementation for Product with the used of PHP Non-framework, Angularjs, Foundation Zurb, MySQL.

To deploy on your localhost. The following are the instructions.

1. Import/Restore the database found in sql folder.
2. Make sure that the folder on your document root are with folder name "myapp". The url will be. http://localhost/myapp

Changing the site prefix:
  Just change the site_prefix variable found in directory test/client/assets/js/boot.js

 API Key are used for authorization. To update the API key, just change the value found in api/config/api.php. 
  Make sure that you will update also the API key on the client side.
